# Phonebook app 
This is the phonebook app, which is a simple application that allows you to store and manage your contacts. It is built using Node.js and Express, and it uses a JSON file to store the contacts data. The app has a simple user interface that allows you to add, edit, and delete contacts, as well as view the list of contacts.

For checking the app, please go to [https://fullstackopen-7uhu.onrender.com/](https://fullstackopen-7uhu.onrender.com/)
